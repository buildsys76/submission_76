To generate this data programmatically (using Python), you can use a tool like NumPy or pandas to create an array of random values within the specified range and constraints.

Here's an example using NumPy:

```python
import numpy as np

# Define constants
BUILDING_ID = 113
YEARLY_MEAN_KWH = 363.97446
SUMMER_MEAN_KWH = 365.49576
WINTER_MEAN_KWH = 361.4717
YEARLY_MIN_KWH = 159.643
YEARLY_MAX_KWH = 679.25

# Calculate number of days in each month for the given year (2016)
days_in_month = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

# Generate random data
np.random.seed(0)  # For reproducibility
values = []
for i in range(720):
    month = (i // days_in_month[i % len(days_in_month)]) + 1
    if month in [6, 7, 8]:  # Summer months (Jun-Aug)
        value = np.random.uniform(YEARLY_MIN_KWH, YEARLY_MAX_KWH) * 0.9 + SUMMER_MEAN_KWH
    elif month in [12, 1, 2]:  # Winter months (Dec-Feb)
        value = np.random.uniform(YEARLY_MIN_KWH, YEARLY_MAX_KWH) * 0.9 + WINTER_MEAN_KWH
    else:  # Other months (Jan-May, Sep-Nov)
        value = np.random.uniform(YEARLY_MIN_KWH, YEARLY_MAX_KWH) * 0.95 + YEARLY_MEAN_KWH
    values.append(value)

# Print the data in the desired format
for i in range(720):
    date_str = f"2016-{int(i // days_in_month[i % len(days_in_month)]):02d}-{((i % len(days_in_month)) + 1):02d} {i % 24:02d}:00"
    value_str = "{:.2f}".format(values[i])
    print(f"{date_str},{value_str}")
```
